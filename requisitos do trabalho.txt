Seria as credenciais do usuário do banco de dados.
Se você está utilizando JDBC para se conectar com o banco de dados, você precisa fornecer as credenciais de acesso ao banco de dados.
A linha de código que faz isso, é algo semelhante a isso:
Connection connection = DriverManager.getConnection("jdbc:postgresql://localhost:5432/bancodedados", "postgres", "ufc123");
As strings do método getConnection podem mudar dependendo de como o postgresql foi instalado. Da mesma forma, o usuário e a senha pode mudar de uma máquina para outra.
Em resumo, seria informar as classes ou arquivos onde se precisa alterar para poder fazer o aplicativo rodar em outra máquina.
